# USELESS_SOFTWARE之HeadPhoto999

*这是一个业余码农练手项目*

* 开发环境：VS2022  
* 开发语言：C#  
* .NET版本：4.7.2 

1. 程序启动后建一个全屏透明窗口
2. 获取窗体的Graphics
3. 使用Graphics.DrawImage()方法，绘制到随机头像
4. 只做上面这些，图片能绘制出来，但是会一闪而过
5. 把绘图的函数绑定到Paint的EventHandler上
6. 把绘图函数放入循环，让随机头像覆盖整个屏幕
